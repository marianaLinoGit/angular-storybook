# Accessibility

O Model UI Kit segue princípios básicos de acessibilidade para garantir melhor experiência para todos os usuários.

---

## Objetivos

- Navegação por teclado
- Contraste adequado
- Estados visuais claros
- Compatibilidade com leitores de tela
- Semântica HTML adequada

## Focus Visible

Todos os componentes interativos devem possuir estado de foco visível.

```scss
:focus-visible {
  outline: 2px solid var(--ui-color-primary);
  outline-offset: 2px;
}
```

---

## Contraste

As cores devem manter boa legibilidade entre:

- Texto e fundo
- Estados de erro
- Estados de sucesso
- Estados desabilitados
- Elementos focados

---

## ARIA

Exemplos de uso:

```html
<button aria-label="Salvar">
  Salvar
</button>
```

```html
<div role="alert">
  Operação concluída com sucesso
</div>
```

---

## Screen Readers

Boas práticas adotadas:

- Uso de HTML semântico
- Labels descritivas
- Hierarquia correta de títulos
- Feedbacks acessíveis
- Botões com texto claro
