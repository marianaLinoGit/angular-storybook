# Model UI Kit

**Model UI Kit** é um projeto de portfólio criado para demonstrar a construção de um pequeno Design System com Angular moderno e Storybook.

O objetivo é documentar, testar visualmente e validar componentes de interface de forma isolada, sem depender de uma aplicação completa.

---

## Objetivo do projeto

Este projeto foi criado para demonstrar:

- Componentização com Angular
- Uso de Storybook como documentação visual
- Componentes customizáveis e reutilizáveis
- Organização baseada em Design System
- Uso de tokens visuais para cores, fontes, espaçamentos e sombras
- Boas práticas com componentes standalone
- Uso de recursos modernos do Angular, como Signals, `input()`, `output()`, `computed()`, `@if` e `@for`

---

## Tecnologias utilizadas

- Angular
- TypeScript
- SCSS
- Storybook
- Angular Standalone Components
- Angular Signals
- Design Tokens com CSS Variables
- Google Fonts
- Puritan para títulos
- Pliant para textos

## Componentes disponíveis

Componente de botão customizável com suporte a:

- Variação de cor
- Tamanho
- Ícone opcional
- Ícone à esquerda ou à direita
- Estado disabled
- Versão outline
- Versão arredondada
- Largura total
- Classe CSS customizada

Componente de alerta com suporte a:

- Mensagem dinâmica
- Ícone opcional
- Fechamento manual
- Variações de cor
- Exibição fixa no topo

### Card

Componente para exibição de conteúdo com suporte a:

- Título

- Descrição
- Imagem opcional
- Footer opcional
- Estado clicável
- Destaque visual
- Borda
- Sombra
- Alinhamento

- Spinner

- Skeleton card
- Skeleton table
- Skeleton avatar
- Overlay

### Modal

Componente de modal com suporte a:

- Modal informativo

- Modal de conteúdo

### Empty State

Componente para estados vazios com suporte a:

- Ícone

- Texto
- Botão opcional

### Accordion

Componente de acordeão com suporte a:

- Múltiplos itens

- Itens desabilitados

### Breadcrumb

Componente de navegação hierárquica com suporte a:

- Separador customizável
- Itens clicáveis
- Itens desabilitados

### Tooltip

Componente de tooltip com suporte a:

- Texto simples
- HTML formatado
- Posição customizável

### Tabs

Componente de abas com suporte a:

- Conteúdo dinâmico

## Design System

O projeto usa tokens globais para manter consistência visual.

Os tokens incluem:

- Fontes
- Tamanhos de fonte
- Pesos de fonte
- Cores
- Espaçamentos
- Bordas
- Raios de borda
- Sombras

Esses tokens ficam centralizados em:

```txt
src/styles/_tokens.scss
```
