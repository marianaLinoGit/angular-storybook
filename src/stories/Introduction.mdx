# Model UI Kit

Design System e biblioteca de componentes desenvolvidos com Angular moderno, Storybook e Design Tokens.

Este projeto foi criado como portfólio técnico para demonstrar arquitetura front-end, componentização, documentação visual, acessibilidade e escalabilidade de interfaces.

---

## Visão Geral

O Model UI Kit foi construído utilizando recursos modernos do Angular e uma abordagem baseada em Design System.

Todos os componentes foram desenvolvidos para serem:

* Reutilizáveis
* Configuráveis
* Documentados
* Escaláveis
* Independentes de aplicação

O objetivo é permitir que equipes desenvolvam interfaces consistentes utilizando os mesmos padrões visuais e comportamentais.

## Design Tokens

Todos os componentes utilizam tokens globais centralizados.

Localização:

```txt
src/styles/_tokens.scss
```

Tokens disponíveis:

* Colors
* Typography
* Font Sizes
* Font Weights
* Spacing
* Border Radius
* Shadows

Exemplo:

```scss
--ui-color-primary
--ui-color-secondary

--ui-font-family-title
--ui-font-family-text

--ui-space-4
--ui-space-6

--ui-radius-md

--ui-shadow-md
```
